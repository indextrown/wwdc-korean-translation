# WWDC 한글 번역 개인정보처리방침

최종 업데이트: 2026-05-30

WWDC 한글 번역은 Apple Developer WWDC 영상 페이지의 영어 자막을 한국어로 번역해 영상 위에 표시하는 Chrome 확장 프로그램입니다.

## 수집 및 사용하는 데이터

이 확장 프로그램은 기능 제공을 위해 다음 데이터를 사용할 수 있습니다.

- WWDC 영상 페이지의 자막 텍스트
- 사용자가 직접 입력한 번역 API 키
- 자막 켜기/끄기, 번역 제공자, 자막 크기, 자막 위치 등 확장 프로그램 설정
- 중복 번역을 줄이기 위한 번역 캐시

## 데이터 사용 목적

데이터는 WWDC 자막 번역과 자막 표시 기능을 제공하기 위해서만 사용됩니다.

- 자막 텍스트는 사용자가 선택한 번역 제공자에 전송되어 한국어 번역 결과를 받는 데 사용될 수 있습니다.
- API 키와 설정은 사용자의 브라우저 로컬 저장소에 저장됩니다.
- 번역 캐시는 같은 자막을 반복 번역하지 않기 위해 사용자의 브라우저 로컬 저장소에 저장됩니다.

## 데이터 저장 위치

API 키, 설정, 번역 캐시는 `chrome.storage.local`에 저장됩니다. 이 확장 프로그램은 별도 자체 서버를 운영하지 않으며, 자체 서버에 사용자 데이터를 저장하지 않습니다.

## 제3자 전송

사용자가 cloud 번역 제공자를 선택한 경우, 번역을 위해 WWDC 자막 텍스트가 해당 번역 제공자 API로 전송될 수 있습니다.

지원 가능한 번역 제공자는 다음과 같습니다.

- Chrome 내장 번역
- Azure AI Translator
- Google Cloud Translation
- DeepL
- NAVER Cloud Papago
- AWS Translate
- LibreTranslate

각 번역 제공자의 데이터 처리 방식은 해당 제공자의 약관과 개인정보처리방침을 따릅니다.

## 판매 및 광고 사용 금지

이 확장 프로그램은 사용자 데이터를 판매하지 않습니다. 또한 광고, 추적, 신용도 판단, 대출 심사, 제3자 마케팅 목적에 사용자 데이터를 사용하거나 전송하지 않습니다.

## 원격 코드

이 확장 프로그램은 원격 코드를 실행하지 않습니다. 확장 프로그램 실행에 필요한 JavaScript 코드는 패키지 안에 포함되어 있습니다.

## 문의

개인정보처리방침에 대한 문의는 Chrome Web Store 등록정보의 지원 경로를 통해 연락할 수 있습니다.
